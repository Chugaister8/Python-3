## Розділ 2. Налаштування Python

Ця книга присвячена програмуванню комп'ютерів на Python. Ви могли б прочитати її від кірки до кірки і засвоїти інформацію, жодного разу не торкнувшись клавіатури, але тоді ви позбавили б себе найцікавішої частини — написання коду.

Щоб отримати максимум від цієї книги, вам потрібен комп'ютер із встановленим Python і спосіб створювати, редагувати та зберігати файли з кодом на Python.

У цьому розділі ви навчитеся:

- встановлювати останню версію Python 3 на свій комп'ютер;
- відкривати IDLE — вбудоване інтегроване середовище розробки та навчання Python.

Розпочнемо!

### 2.1 Примітка щодо версій Python

Багато операційних систем, таких як macOS і Linux, постачаються з попередньо встановленим Python. Версія Python, що постачається разом із вашою операційною системою, називається системним Python.

Системний Python майже завжди застарілий і може навіть не бути повноцінною інсталяцією Python. Вкрай важливо мати найновішу версію Python, щоб успішно виконувати приклади з цієї книги.

На вашому комп'ютері може бути встановлено кілька версій Python одночасно. У цьому розділі ви встановите найновішу версію Python 3 поряд із будь-яким системним Python, що вже може існувати на вашій машині.

> **Примітка**
>
> Навіть якщо у вас уже встановлено Python 3.8, все одно варто побіжно переглянути цей розділ, щоб переконатися, що ваше середовище налаштоване для роботи з книгою.

Цей розділ поділено на три частини: Windows, macOS та Ubuntu Linux. Знайдіть частину для вашої операційної системи і виконайте кроки для налаштування, після чого переходьте до наступного розділу.

Якщо у вас інша операційна система, зверніться до посібника Real Python «Python 3 Installation & Setup Guide», щоб дізнатися, чи охоплено вашу ОС.

### 2.2 Windows

Виконайте наступні кроки для встановлення Python 3 і відкриття IDLE у Windows.

> **Важливо**
>
> Код у цій книзі протестовано лише з Python, встановленим відповідно до інструкцій цього розділу.
>
> Майте на увазі: якщо ви встановили Python іншим способом, наприклад через Anaconda Python, під час виконання прикладів коду можуть виникнути проблеми.

#### Встановлення Python

У системах Windows Python зазвичай не встановлений заздалегідь. На щастя, встановлення не потребує нічого більшого, ніж завантаження інсталятора Python з сайту python.org і його запуску.

**Крок 1: Завантажте інсталятор Python 3**

Відкрийте браузер і перейдіть на сторінку завантажень для Windows на python.org.

Під заголовком «Python Releases for Windows» натисніть на посилання для останнього випуску Python 3 — Python 3.x.x. На момент написання книги остання версія — Python 3.8. Потім прокрутіть сторінку вниз і виберіть «Windows x86-64 executable installer».

> **Примітка**
>
> Якщо у вашій системі 32-розрядний процесор, виберіть 32-розрядний інсталятор. Якщо ви не впевнені, чи є ваш комп'ютер 32- або 64-розрядним, скористайтеся вказаним вище 64-розрядним інсталятором.

**Крок 2: Запустіть інсталятор**

Запустіть інсталятор, двічі клацнувши на завантаженому файлі. Ви побачите таке вікно:

> **Важливо**
>
> Обов'язково встановіть прапорець «Add Python 3.x to PATH», як показано, щоб під час встановлення інтерпретатор було додано до шляху виконання.
>
> Якщо ви встановили Python без цього прапорця, можна запустити інсталятор повторно і встановити його.

Натисніть «Install Now» для встановлення Python 3. Зачекайте завершення встановлення, після чого переходьте до відкриття IDLE.

#### Відкриття IDLE

IDLE можна відкрити у два кроки:

1. Натисніть на меню «Пуск» і знайдіть папку Python 3.8.
2. Відкрийте папку і виберіть IDLE (Python 3.8).

> **Примітка**
>
> Рекомендуємо використовувати IDLE для роботи з цією книгою.
>
> За бажанням ви можете використовувати інший редактор коду. Однак деякі розділи, наприклад Розділ 7: Пошук і виправлення помилок у коді, містять матеріал, специфічний для IDLE.

IDLE відкриває оболонку Python у новому вікні. Оболонка Python — це інтерактивне середовище, що дозволяє вводити код Python і негайно виконувати його. Це чудовий спосіб розпочати роботу з Python!

Вікно оболонки Python виглядає так:

У верхній частині вікна відображаються версія Python і деяка інформація про операційну систему. Якщо ви бачите версію нижче ніж 3.7, можливо, вам потрібно повторити інструкції з установки з попереднього розділу.

Символ `>>>`, який ви бачите, називається запрошенням (prompt). Щоразу, коли ви його бачите, це означає, що Python очікує від вас інструкцій.

### 2.3 macOS

Виконайте наступні кроки для встановлення Python 3 і відкриття IDLE на macOS.

> **Важливо**
>
> Код у цій книзі протестовано лише з Python, встановленим відповідно до інструкцій цього розділу.
>
> Майте на увазі: якщо ви встановили Python іншим способом, наприклад через Homebrew або Anaconda Python, під час виконання прикладів коду можуть виникнути проблеми.

#### Встановлення Python

На більшості машин з macOS встановлено Python 2. Вам потрібно встановити останню версію Python 3. Це можна зробити, завантаживши інсталятор із сайту python.org.

**Крок 1: Завантажте інсталятор Python 3**

Відкрийте браузер і перейдіть на сторінку завантажень для macOS на python.org.

Під заголовком «Python Releases for macOS» натисніть на посилання для останнього випуску Python 3 — Python 3.x.x. На момент написання книги остання версія — Python 3.8. Потім прокрутіть сторінку вниз і виберіть «macOS 64-bit/32-bit installer». Розпочнеться завантаження.

**Крок 2: Запустіть інсталятор**

Запустіть інсталятор, двічі клацнувши на завантаженому файлі. Ви побачите таке вікно:

1. Кілька разів натисніть кнопку «Continue», доки вас не попросять погодитися з ліцензійною угодою. Потім натисніть «Agree». Відобразиться вікно, що повідомляє, куди буде встановлено Python і скільки місця це займе.

2. Швидше за все, ви не захочете змінювати стандартне розташування, тому натисніть «Install» для початку встановлення. Інсталятор Python повідомить вас, коли копіювання файлів завершиться.

3. Натисніть «Close», щоб закрити вікно інсталятора. Тепер, коли Python встановлено, можна відкрити IDLE і підготуватися до написання першої програми на Python.

#### Відкриття IDLE

IDLE можна відкрити у три кроки:

1. Відкрийте Finder і натисніть на «Програми» (Applications).
2. Знайдіть папку Python 3.8 і двічі клацніть на ній.
3. Двічі клацніть на значку IDLE.

Також можна відкрити IDLE за допомогою функції пошуку Spotlight. Натисніть `Cmd+Пробіл`, щоб відкрити пошук Spotlight, введіть слово `idle`, потім натисніть `Return`, щоб відкрити IDLE.

>
> За бажанням ви можете використовувати інший редактор коду. Однак деякі розділи, наприклад Розділ 7: Пошук і виправлення помилок у коді, містять матеріал, специфічний для IDLE.

### 2.4 Ubuntu Linux

Виконайте наступні кроки для встановлення Python 3 і відкриття IDLE на Ubuntu Linux.

> **Важливо**
>
> Код у цій книзі протестовано лише з Python, встановленим відповідно до інструкцій цього розділу.
>
> Майте на увазі: якщо ви встановили Python іншим способом, наприклад через Anaconda Python, під час виконання прикладів коду можуть виникнути проблеми.

#### Встановлення Python

Цілком можливо, що у вашому дистрибутиві Ubuntu Python уже встановлено, але, швидше за все, це не остання версія, і це може бути Python 2, а не Python 3.

Щоб дізнатися, яка версія (або версії) у вас встановлена, відкрийте вікно терміналу і спробуйте такі команди:

```
$ python --version
$ python3 --version
```

Одна або кілька з цих команд мають вивести версію, як показано нижче (номер вашої версії може відрізнятися):

```
$ python3 --version
Python 3.8.1
```

Якщо відображається Python 2.x або версія Python 3 нижче ніж 3.8, вам потрібно встановити найновішу версію. Спосіб встановлення Python на Ubuntu залежить від того, яка версія Ubuntu у вас встановлена. Щоб дізнатися локальну версію Ubuntu, виконайте таку команду:

```
$ lsb_release -a
No LSB modules are available.
Distributor ID: Ubuntu
Description:    Ubuntu 18.04.1 LTS
Release:        18.04
Codename:       bionic
```

Подивіться на номер версії поруч із «Release» у виведенні консолі та дотримуйтесь відповідних інструкцій нижче.

**Ubuntu 18.04+**

У версії Ubuntu 18.04 Python 3.8 не встановлено за замовчуванням, але він є в репозиторії Universe. Встановити його можна такими командами у додатку «Термінал»:

```
$ sudo apt-get update
$ sudo apt-get install python3.8 idle-python3.8
```

**Ubuntu 17 та нижче**

Для версій Ubuntu 17 і нижче Python 3.8 відсутній у репозиторії Universe. Його потрібно отримати з Personal Package Archive (PPA). Щоб встановити Python із PPA «deadsnakes», виконайте такі команди у додатку «Термінал»:

```
$ sudo add-apt-repository ppa:deadsnakes/ppa
$ sudo apt-get update
$ sudo apt-get install python3.8 idle-python3.8
```

Перевірити, чи встановлено правильну версію Python, можна командою `python3 --version`. Якщо ви бачите версію нижче ніж 3.7, можливо, вам потрібно ввести `python3.8 --version`. Тепер ви готові відкрити IDLE і підготуватися до написання першої програми на Python.

#### Відкриття IDLE

IDLE можна відкрити з командного рядка, ввівши:

```
$ idle-python3.8
```

На деяких установках Linux IDLE можна відкрити скороченою командою:

```
$ idle3
```

> **Примітка**
>
> Рекомендуємо використовувати IDLE для роботи з цією книгою.
>
> За бажанням ви можете використовувати інший редактор коду. Однак деякі розділи, наприклад Розділ 7: Пошук і виправлення помилок у коді, містять матеріал, специфічний для IDLE.

IDLE відкриває оболонку Python у новому вікні. Оболонка Python — це інтерактивне середовище, що дозволяє вводити код Python і негайно виконувати його. Це чудовий спосіб розпочати роботу з Python!

> **Важливо**

> Якщо ви відкрили IDLE командою `idle3` і у вікні оболонки Python бачите версію нижче ніж 3.7, вам потрібно відкрити IDLE командою `idle-python3.8`.
