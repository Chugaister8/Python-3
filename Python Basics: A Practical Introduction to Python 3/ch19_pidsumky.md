# Розділ 19. Підсумкові думки та наступні кроки

Вітаємо! Ви дісталися кінця цієї книги.

Ви вже знаєте достатньо, щоб робити багато дивовижних речей із Python, але тепер починається справжня радість: час досліджувати самостійно!

Найкращий спосіб навчитися — вирішувати реальні задачі. Ваш код може бути не дуже красивим на початку, але він буде корисним. Якщо ви не думаєте, що маєте задачі, які Python міг би вирішити, — оберіть популярний модуль, що вас цікавить, і створіть власний проєкт навколо нього.

Частина того, що робить Python таким чудовим, — це спільнота. Знаєте когось, хто вивчає Python? Допоможіть йому! Єдиний спосіб дійсно знати, що ви опанували концепцію, — це коли можете пояснити її комусь іншому.

Далі поглибтесь у більш просунуті матеріали на realpython.com або переглядайте статті в розсилці PyCoder's Weekly.

Коли відчуєте готовність — розгляньте участь у проєктах із відкритим кодом на GitHub. Якщо більше до вподоби головоломки — спробуйте математичні задачі на Project Euler.

Якщо десь застрягнете, майже гарантовано, що хтось інший уже стикався (і можливо вирішив) точно таку саму проблему — шукайте відповіді на Stack Overflow або в спільноті Python.
