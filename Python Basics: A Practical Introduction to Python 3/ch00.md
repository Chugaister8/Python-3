## Про авторів

У Real Python ви навчаєтеся практичним навичкам програмування від спільноти досвідчених Pythonistas з усього світу. Сайт realpython.com запрацював у 2012 році і на сьогодні щомісяця допомагає понад двом мільйонам Python-розробників безкоштовними навчальними матеріалами та поглибленими навчальними ресурсами.

Кожен, хто працював над цією книгою, є практикуючим фахівцем із кількарічним досвідом роботи в індустрії програмного забезпечення. Ось члени команди Real Python Tutorial Team, які працювали над Python Basics:

**Fletcher Heisler** — засновник Hunter2, де він навчає розробників злому та захисту сучасних вебзастосунків. Як один із засновників Real Python, Fletcher написав оригінальну версію цієї книги у 2012 році.

**David Amos** — математик за освітою, дата-сайентист / Python-розробник за фахом і любитель кави за покликанням. Він є членом команди Real Python і переписав значні частини цієї книги, оновивши її до Python 3.

**Dan Bader** — власник і головний редактор Real Python, пристрасний шанувальник Python. Коли він не зайнятий роботою над навчальною платформою Real Python, то допомагає Python-розробникам виводити свої навички програмування на новий рівень за допомогою навчальних матеріалів, книг та онлайн-курсів.

**Joanna Jablonski** — виконавчий редактор Real Python. Вона так само любить природні мови, як і мови програмування. Коли вона не створює навчальні матеріали для Python-розробників, то шукає нові способи оптимізувати різні аспекти свого життя.
