## Розділ 1. Вступ

Ласкаво просимо до книги Python Basics від Real Python, повністю оновленої для Python 3.8! У цій книзі ви навчитеся практичним методам програмування на Python, проілюстрованим корисними та цікавими прикладами.

Незалежно від того, чи ви новачок у програмуванні, чи досвідчений розробник, який хоче освоїти нову мову, ця книга навчить вас усього практичного Python, необхідного для того, щоб самостійно розпочати власні проєкти.

Незалежно від ваших кінцевих цілей, якщо ви взагалі працюєте з комп'ютером, ви невдовзі знаходитимете безліч способів покращити своє життя, автоматизуючи завдання та вирішуючи проблеми за допомогою програм на Python, які ви самі створюєте.

Але чим так хороший Python як мова програмування? Python — це вільне програмне забезпечення з відкритим кодом: ви можете завантажити його безкоштовно і використовувати для будь-яких цілей — комерційних чи ні.

Python також має дивовижну спільноту, яка створила низку корисних додаткових інструментів для використання у власних програмах. Потрібно працювати з PDF-документами? Для цього є повноцінний інструмент. Хочете збирати дані з вебсторінок? Починати з нуля не потрібно!

Python створений таким чином, щоб бути простішим у використанні, ніж інші мови програмування. Зазвичай код на Python значно легше читати і значно швидше писати, ніж на інших мовах.

Наприклад, ось простий код мовою C — ще однією широко використовуваною мовою програмування:

```c
#include <stdio.h>

int main(void)
{
printf("Hello, world\n");
}
```

Ця програма лише виводить текст `Hello, world` на екран. Скільки роботи заради одного рядка! Ось та сама програма, написана на Python:

```python
print("Hello, world")
```

Досить просто, правда? Код на Python пишеться швидше і читається легше. Нам здається, він виглядає привітнішим і доступнішим!

При цьому Python має всю функціональність інших мов і навіть більше. Ви можете здивуватися, скільки професійних продуктів побудовано на коді Python: Instagram, YouTube, Reddit, Spotify — щоб назвати лише деякі.

Python — це не лише дружня та цікава мова для вивчення: вона також забезпечує роботу технологій багатьох компаній світового класу та пропонує чудові кар'єрні можливості для будь-якого програміста, який опанує її.
